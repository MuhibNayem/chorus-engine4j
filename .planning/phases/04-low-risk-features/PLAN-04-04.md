# Plan 04-04: Automated Evaluator Triggers

## Goal
Automatically trigger evaluators when a run reaches a terminal state (SUCCESS or ERROR). Uses Spring's `ApplicationEvent` mechanism: `OtlpIngestionService` publishes `RunCompletedEvent` from `flushRun()`, and a listener queries all `kind="hallucination"` evaluators and invokes `EvaluatorService.evaluateRun()` for each.

## must_haves
- `RunCompletedEvent` extends `ApplicationEvent` with `runId`, `tenantId`, `status`.
- `OtlpIngestionService.flushRun()` publishes event exactly once per run when terminal state is first reached.
- `RunCompletedEventListener` uses `@EventListener`, queries evaluators via `EvaluatorRepository.findByKind("hallucination")`, calls `EvaluatorService.evaluateRun()`.
- No async executor — synchronous handling keeps ordering simple.
- Tests cover: event published once, listener triggers evaluator, no trigger on non-terminal state.

```java
package com.chorus.observe.event;

import com.chorus.observe.model.Run;
import org.jspecify.annotations.NonNull;
import org.springframework.context.ApplicationEvent;

import java.util.Objects;

public class RunCompletedEvent extends ApplicationEvent {

    private final @NonNull String runId;
    private final @NonNull String tenantId;
    private final @NonNull Run.Status status;

    public RunCompletedEvent(@NonNull Object source, @NonNull String runId,
                             @NonNull String tenantId, @NonNull Run.Status status) {
        super(source);
        this.runId = Objects.requireNonNull(runId);
        this.tenantId = Objects.requireNonNull(tenantId);
        this.status = Objects.requireNonNull(status);
    }

    public @NonNull String runId() { return runId; }
    public @NonNull String tenantId() { return tenantId; }
    public @NonNull Run.Status status() { return status; }
}
```
</action>

## Task 3: Create RunCompletedEventListener

<action>
Create `chorus-observe-server/src/main/java/com/chorus/observe/event/RunCompletedEventListener.java`:

```java
package com.chorus.observe.event;

import com.chorus.observe.model.Evaluator;
import com.chorus.observe.persistence.EvaluatorRepository;
import com.chorus.observe.service.EvaluatorService;
import org.jspecify.annotations.NonNull;
import org.slf4j.Logger;
import org.slf4j.LoggerFactory;
import org.springframework.context.event.EventListener;
import org.springframework.stereotype.Component;

import java.util.List;
import java.util.Objects;

@Component
public class RunCompletedEventListener {

    private static final Logger LOG = LoggerFactory.getLogger(RunCompletedEventListener.class);

    private final EvaluatorRepository evaluatorRepository;
    private final EvaluatorService evaluatorService;

    public RunCompletedEventListener(@NonNull EvaluatorRepository evaluatorRepository,
                                     @NonNull EvaluatorService evaluatorService) {
        this.evaluatorRepository = Objects.requireNonNull(evaluatorRepository);
        this.evaluatorService = Objects.requireNonNull(evaluatorService);
    }

    @EventListener
    public void onRunCompleted(@NonNull RunCompletedEvent event) {
        LOG.debug("Run completed: {} (status={})", event.runId(), event.status());

        List<Evaluator> evaluators = evaluatorRepository.findByKind("hallucination");
        for (Evaluator evaluator : evaluators) {
            try {
                evaluatorService.evaluateRun(event.runId(), evaluator.evaluatorId());
                LOG.debug("Triggered evaluator {} for run {}", evaluator.evaluatorId(), event.runId());
            } catch (Exception e) {
                LOG.error("Failed to trigger evaluator {} for run {}", evaluator.evaluatorId(), event.runId(), e);
            }
        }
    }
}
```

Register bean in `ChorusObserveAutoConfiguration` with `@ConditionalOnMissingBean`.
</action>
