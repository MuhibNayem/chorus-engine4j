# Plan 02-03: SAML 2.0

## Goal
Implement SP-initiated SAML 2.0 single sign-on using Spring Security 7.0.0's `saml2Login()`. Supports Okta, Azure AD, and ADFS. Per-tenant SAML configs (entity ID, signing cert thumbprint, ACS URL) are stored in `tenant_saml_configs` and loaded at request time. Assertion replay attacks are blocked via an in-memory assertion ID cache with 2-minute TTL.

## must_haves
- Spring Security `saml2Login()` is configured in the SecurityFilterChain.
- A custom `RelyingPartyRegistrationRepository` loads registrations from `tenant_saml_configs`.
- `MetadataResolver` fetches IdP metadata from the configured URL and resolves the X.509 certificate by thumbprint.
- `Saml2AuthenticationSuccessHandler` extracts SAML assertion attributes, JIT-provisions users, and issues a Chorus JWT.
- `AssertionIdCache` stores seen assertion IDs with 2-minute expiration; duplicate assertions within the window are rejected.
- Admin can CRUD SAML configs via `/api/v1/admin/saml-configs`.
- `./gradlew :chorus-observe-server:compileJava` passes.

```java
package com.chorus.observe.security.saml2;

import org.jspecify.annotations.NonNull;

import java.time.Duration;
import java.time.Instant;
import java.util.concurrent.ConcurrentHashMap;

/**
 * In-memory cache of seen SAML assertion IDs to prevent replay attacks.
 * Entries expire after a fixed TTL (2 minutes).
 */
public class AssertionIdCache {

    private static final Duration TTL = Duration.ofMinutes(2);
    private final ConcurrentHashMap<String, Instant> cache = new ConcurrentHashMap<>();

    /**
     * Checks if the assertion ID has been seen within the TTL window.
     * If not seen, records it and returns false.
     * If seen and not expired, returns true (replay detected).
     */
    public synchronized boolean isReplay(@NonNull String assertionId) {
        Instant now = Instant.now();
        // Clean expired entries opportunistically
        cache.entrySet().removeIf(e -> e.getValue().isBefore(now));

        Instant seenAt = cache.get(assertionId);
        if (seenAt != null) {
            return true; // replay
        }
        cache.put(assertionId, now.plus(TTL));
        return false;
    }
}
```

This class fetches SAML IdP metadata from a URL and extracts the X.509 signing certificate by SHA-256 thumbprint.

```java
package com.chorus.observe.security.saml2;

import org.jspecify.annotations.NonNull;
import org.jspecify.annotations.Nullable;

import java.net.URI;
import java.net.http.HttpClient;
import java.net.http.HttpRequest;
import java.net.http.HttpResponse;
import java.security.MessageDigest;
import java.security.cert.CertificateFactory;
import java.security.cert.X509Certificate;
import java.time.Duration;
import java.util.Base64;

/**
 * Fetches SAML IdP metadata and resolves X.509 certificates by thumbprint.
 */
public class MetadataResolver {

    private final HttpClient httpClient;

    public MetadataResolver() {
        this.httpClient = HttpClient.newBuilder()
            .connectTimeout(Duration.ofSeconds(10))
            .build();
    }

    /**
     * Fetches the IdP metadata from the given URL, parses the X.509 certificates,
     * and returns the certificate whose SHA-256 thumbprint matches.
     */
    public @Nullable String resolveCertificatePem(@NonNull String metadataUrl,
                                                   @NonNull String expectedThumbprint) {
        try {
            HttpRequest request = HttpRequest.newBuilder()
                .uri(URI.create(metadataUrl))
                .timeout(Duration.ofSeconds(10))
                .GET()
                .build();

            HttpResponse<String> response = httpClient.send(request, HttpResponse.BodyHandlers.ofString());
            if (response.statusCode() != 200) {
                return null;
            }

            String body = response.body();
            // Extract all <X509Certificate> elements from the metadata XML
            // Simple regex-based extraction — sufficient for standard metadata
            java.util.regex.Pattern pattern = java.util.regex.Pattern.compile(
                "<X509Certificate>([^<]+)</X509Certificate>");
            java.util.regex.Matcher matcher = pattern.matcher(body);

            while (matcher.find()) {
                String base64Cert = matcher.group(1).replaceAll("\\s", "");
                String thumbprint = computeThumbprint(base64Cert);
                if (thumbprint.equalsIgnoreCase(expectedThumbprint)) {
                    return "-----BEGIN CERTIFICATE-----\n"
                        + base64Cert.replaceAll("(.{64})", "$1\n")
                        + "\n-----END CERTIFICATE-----";
                }
            }
            return null;
        } catch (Exception e) {
            return null;
        }
    }

    private @NonNull String computeThumbprint(@NonNull String base64Cert) throws Exception {
        byte[] certBytes = Base64.getDecoder().decode(base64Cert);
        CertificateFactory factory = CertificateFactory.getInstance("X.509");
        X509Certificate cert = (X509Certificate) factory.generateCertificate(
            new java.io.ByteArrayInputStream(certBytes));
        byte[] digest = MessageDigest.getInstance("SHA-256").digest(cert.getEncoded());
        return bytesToHex(digest);
    }

    private @NonNull String bytesToHex(byte[] bytes) {
        StringBuilder sb = new StringBuilder();
        for (byte b : bytes) {
            sb.append(String.format("%02x", b));
        }
        return sb.toString();
    }
}
```

**Note:** This uses a simple regex-based XML extraction. For production robustness, consider using OpenSAML's `MetadataResolver` class, but that adds complexity. The regex approach works for standard IdP metadata from Okta/Azure AD.
</action>

```java
package com.chorus.observe.security.saml2;

import com.chorus.observe.model.TenantSamlConfig;
import com.chorus.observe.persistence.TenantSamlConfigRepository;
import org.jspecify.annotations.NonNull;
import org.springframework.core.io.ByteArrayResource;
import org.springframework.security.saml2.core.Saml2X509Credential;
import org.springframework.security.saml2.provider.service.registration.RelyingPartyRegistration;
import org.springframework.security.saml2.provider.service.registration.RelyingPartyRegistrationRepository;
import org.springframework.security.saml2.provider.service.registration.Saml2MessageBinding;

import java.nio.charset.StandardCharsets;
import java.security.cert.X509Certificate;
import java.security.interfaces.RSAPrivateKey;

public class TenantSamlConfigRelyingPartyRegistrationRepository implements RelyingPartyRegistrationRepository {

    private final TenantSamlConfigRepository configRepository;
    private final MetadataResolver metadataResolver;

    public TenantSamlConfigRelyingPartyRegistrationRepository(
            @NonNull TenantSamlConfigRepository configRepository,
            @NonNull MetadataResolver metadataResolver) {
        this.configRepository = configRepository;
        this.metadataResolver = metadataResolver;
    }

    @Override
    public RelyingPartyRegistration findByRegistrationId(@NonNull String registrationId) {
        // registrationId format: "{tenantId}__{providerName}"
        int sep = registrationId.indexOf("__");
        if (sep < 0) {
            return null;
        }
        String tenantId = registrationId.substring(0, sep);
        String providerName = registrationId.substring(sep + 2);

        return configRepository.findByTenantIdAndProviderName(tenantId, providerName)
            .filter(TenantSamlConfig::enabled)
            .map(this::toRelyingPartyRegistration)
            .orElse(null);
    }

    private RelyingPartyRegistration toRelyingPartyRegistration(@NonNull TenantSamlConfig config) {
        String registrationId = config.tenantId() + "__" + config.providerName();

        String certPem = metadataResolver.resolveCertificatePem(
            config.metadataUrl(), config.signingCertThumbprint());

        if (certPem == null) {
            throw new IllegalStateException("Could not resolve SAML signing certificate for " + registrationId);
        }

        Saml2X509Credential verificationCredential = Saml2X509Credential.verification(
            loadCertificate(certPem));

        // For SP-initiated SSO, we need a signing credential (self-signed is fine for this use case)
        // In production, use a real key pair. For now, generate an ephemeral key or load from config.
        // TODO: executor should implement key generation or config-based key loading
        RSAPrivateKey privateKey = generateEphemeralKey(); // placeholder
        Saml2X509Credential signingCredential = Saml2X509Credential.signing(privateKey, loadCertificate(certPem));

        return RelyingPartyRegistration.withRegistrationId(registrationId)
            .entityId(config.entityId())
            .assertionConsumerServiceLocation(config.acsUrl())
            .assertionConsumerServiceBinding(Saml2MessageBinding.POST)
            .signingX509Credentials(c -> c.add(signingCredential))
            .decryptionX509Credentials(c -> c.add(signingCredential))
            .assertingPartyDetails(party -> party
                .entityId(config.entityId())
                .singleSignOnServiceLocation(config.signOnUrl())
                .singleSignOnServiceBinding(Saml2MessageBinding.REDIRECT)
                .verificationX509Credentials(c -> c.add(verificationCredential))
            )
            .build();
    }

    private X509Certificate loadCertificate(@NonNull String pem) {
        // Use CertificateFactory to parse PEM
        try {
            CertificateFactory factory = CertificateFactory.getInstance("X.509");
            return (X509Certificate) factory.generateCertificate(
                new java.io.ByteArrayInputStream(pem.getBytes(StandardCharsets.UTF_8)));
        } catch (Exception e) {
            throw new RuntimeException("Failed to load certificate", e);
        }
    }

    private RSAPrivateKey generateEphemeralKey() {
        // TODO: Implement proper key generation or config-based key loading
        // For now, this is a placeholder that the executor must replace
        throw new UnsupportedOperationException("Ephemeral key generation not yet implemented");
    }
}
```

**Important:** The executor MUST replace `generateEphemeralKey()` with a real implementation. Options:
1. Generate a self-signed RSA key pair on startup and cache it.
2. Load a private key from `chorus.observe.saml.sp-private-key` config property.
3. Use `java.security.KeyPairGenerator` to generate a 2048-bit RSA key pair.

The signing credential is needed for SP-initiated SAML AuthNRequest signing. If the IdP does not require signed requests, this can be simplified.

```java
package com.chorus.observe.security.saml2;

import com.chorus.observe.model.User;
import com.chorus.observe.security.JwtTokenService;
import com.chorus.observe.security.oauth2.JitProvisioningService;
import jakarta.servlet.ServletException;
import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import org.jspecify.annotations.NonNull;
import org.springframework.security.core.Authentication;
import org.springframework.security.saml2.provider.service.authentication.Saml2Authentication;
import org.springframework.security.web.authentication.AuthenticationSuccessHandler;

import java.io.IOException;

public class ChorusSaml2AuthenticationSuccessHandler implements AuthenticationSuccessHandler {

    private final JitProvisioningService jitProvisioningService;
    private final JwtTokenService jwtTokenService;
    private final AssertionIdCache assertionIdCache;
    private final String frontendRedirectUrl;

    public ChorusSaml2AuthenticationSuccessHandler(
            @NonNull JitProvisioningService jitProvisioningService,
            @NonNull JwtTokenService jwtTokenService,
            @NonNull AssertionIdCache assertionIdCache,
            @NonNull String frontendRedirectUrl) {
        this.jitProvisioningService = jitProvisioningService;
        this.jwtTokenService = jwtTokenService;
        this.assertionIdCache = assertionIdCache;
        this.frontendRedirectUrl = frontendRedirectUrl;
    }

    @Override
    public void onAuthenticationSuccess(@NonNull HttpServletRequest request,
                                        @NonNull HttpServletResponse response,
                                        @NonNull Authentication authentication)
            throws IOException, ServletException {

        if (!(authentication instanceof Saml2Authentication samlAuth)) {
            response.sendError(HttpServletResponse.SC_BAD_REQUEST, "Invalid SAML authentication");
            return;
        }

        // Extract assertion ID for replay check
        String assertionId = extractAssertionId(samlAuth);
        if (assertionId != null && assertionIdCache.isReplay(assertionId)) {
            response.sendError(HttpServletResponse.SC_BAD_REQUEST, "SAML assertion replay detected");
            return;
        }

        String registrationId = samlAuth.getRelyingPartyRegistrationId();
        String[] parts = registrationId.split("__", 2);
        if (parts.length != 2) {
            response.sendError(HttpServletResponse.SC_BAD_REQUEST, "Invalid registration ID");
            return;
        }
        String tenantId = parts[0];
        String providerName = parts[1];

        // Extract attributes from SAML assertion
        String email = samlAuth.getPrincipal().getName();
        String displayName = extractAttribute(samlAuth, "displayName");
        if (displayName == null) {
            displayName = extractAttribute(samlAuth, " givenname") + " " + extractAttribute(samlAuth, "surname");
        }

        if (email == null || email.isBlank()) {
            response.sendError(HttpServletResponse.SC_BAD_REQUEST, "Email not provided by SAML IdP");
            return;
        }

        // TODO: fetch defaultRole from tenant_saml_configs
        User user = jitProvisioningService.provisionOrLink(
            tenantId, email, displayName, User.AuthSource.SAML, "VIEWER");

        String chorusJwt = jwtTokenService.generateToken(user.userId(), tenantId, java.util.List.of());
        response.sendRedirect(frontendRedirectUrl + "?token=" + chorusJwt);
    }

    private @Nullable String extractAssertionId(@NonNull Saml2Authentication auth) {
        // The assertion ID is available in the Saml2Authentication details
        // In Spring Security 7, this may require casting to OpenSAML types
        // TODO: executor should implement proper assertion ID extraction using OpenSAML Assertion#getID()
        Object details = auth.getDetails();
        if (details instanceof org.opensaml.saml.saml2.core.Assertion assertion) {
            return assertion.getID();
        }
        return null;
    }

    private @Nullable String extractAttribute(@NonNull Saml2Authentication auth, @NonNull String name) {
        Object attribute = auth.getPrincipal().getAttribute(name);
        if (attribute instanceof java.util.List<?> list && !list.isEmpty()) {
            return list.get(0).toString();
        }
        return attribute != null ? attribute.toString() : null;
    }
}
```

**Note:** The assertion ID extraction may need refinement based on the exact OpenSAML 4 API. The executor should test with a real SAML response. The `extractAssertionId` method is a starting point.
</action>
