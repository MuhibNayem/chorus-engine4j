# Plan 01-02: @RequirePermission Annotation and PermissionInterceptor

## Goal
Replace the broken `RbacAuthorizationFilter` (which reads a non-existent `X-Required-Permission` header) with a correct `PermissionInterceptor` that reads `@RequirePermission` annotations from handler methods, checks `TenantContext` scopes, and rejects unauthorized requests with 403. The interceptor is registered via `WebMvcConfigurer`.

## must_haves
- `@RequirePermission(String value)` annotation exists and can be applied to methods.
- `PermissionInterceptor` implements `HandlerInterceptor` and checks scopes against the annotation value in `preHandle()`.
- `PermissionInterceptor` allows access when the user has the required permission OR the `admin` scope.
- `PermissionInterceptor` rejects with 401 when `TenantContext.isAuthenticated()` is false.
- `PermissionInterceptor` rejects with 403 when the authenticated user lacks the required permission.
- `RbacAuthorizationFilter.java` is deleted.
- `@PreAuthorize("hasAuthority('admin')")` remains on admin controllers for defense-in-depth.

import java.lang.annotation.ElementType;
import java.lang.annotation.Retention;
import java.lang.annotation.RetentionPolicy;

/**

}

Create new file `chorus-observe-server/src/main/java/com/chorus/observe/security/PermissionInterceptor.java` with the following exact content:

```java
package com.chorus.observe.security;

import jakarta.servlet.http.HttpServletRequest;
import jakarta.servlet.http.HttpServletResponse;
import org.jspecify.annotations.NonNull;
import org.springframework.web.method.HandlerMethod;
import org.springframework.web.servlet.HandlerInterceptor;

import java.util.Set;

/**
 * Enforces {@link RequirePermission} annotations on controller methods.
 * <p>
 * Runs after the DispatcherServlet resolves the handler, so it has direct
 * access to the annotated method. Checks the {@code scopes} request attribute
 * populated by {@link JwtAuthFilter} or {@link ApiKeyAuthFilter}.
 */
public class PermissionInterceptor implements HandlerInterceptor {

    @Override
    public boolean preHandle(@NonNull HttpServletRequest request, @NonNull HttpServletResponse response,
                             @NonNull Object handler) throws Exception {
        if (!(handler instanceof HandlerMethod handlerMethod)) {
            return true;
        }

        RequirePermission annotation = handlerMethod.getMethodAnnotation(RequirePermission.class);
        if (annotation == null) {
            return true;
        }

        if (!TenantContext.isAuthenticated()) {
            response.sendError(HttpServletResponse.SC_UNAUTHORIZED, "Authentication required");
            return false;
        }

        @SuppressWarnings("unchecked")
        Set<String> scopes = (Set<String>) request.getAttribute("scopes");
        if (scopes == null) {
            scopes = Set.of();
        }

        String required = annotation.value();
        if (!scopes.contains(required) && !scopes.contains(Permission.ADMIN)) {
            response.sendError(HttpServletResponse.SC_FORBIDDEN, "Insufficient permissions: " + required);
            return false;
        }

        return true;
    }
}
```
</action>
